Insights & Research — section options

Layout directions for the homepage Insights section. Neutral/blue palette on purpose so the comparison is about layout; the winner gets themed to each version (F black · G navy · H green) afterward.

Content model:  Blog posts = 16:9 photo + headline + teaser + date.  Insights = digital documents, cover imagery 850×1100 (portrait) + headline + optional teaser, often no date.  Each type shows the first few items with imagery, then older items as a clickable headline stack.

Links are non-functional placeholders. Photos are Unsplash farmland with a gradient fallback if blocked; insight covers stand in for 850×1100 document art.
