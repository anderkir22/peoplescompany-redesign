Homepage Concepts for Review

Seven homepage directions to review. Each is a full, higher-fidelity homepage built on the 2026 design system — color (blue #00AEEF + dark green #005F5B), real photography, and the agreed must-haves: the four-path audience router, a "What We Do" services section, the FarmWorth block with Request a Demo, dual-path CTAs, and a consistent footer. All share the design system at 1400px width and are colorblind-safe (no red). Links open in a new tab.

Shared component

“Find Properties” tab — design options

Every concept opens the property/auction map from a pull-down “Find Properties” tab below the navigation. This page compares the ten visual treatments we’ve explored for that tab — shape, color, and motion — so you can pick one direction to standardize across the whole site. Worth reviewing on its own, separate from the homepage layouts.
