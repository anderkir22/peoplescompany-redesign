Top navigation — structure options

A catalog to compare and shortlist header directions — this pass is purely about layout structure. Each option arranges the same required pieces in a genuinely different way (logo placement, row count, menu alignment, and where the phone · Expo · FarmWorth sit).

Static only — no interactivity here. No hover pop-downs and no sign-in behavior; those get designed separately and will look nothing like a placeholder would. The ▾ on each menu just marks that a sub-nav pop-down exists. Shown in one neutral palette on purpose so you compare layout, not color — any structure can later take the green / navy / gradient treatments. Colorblind-safe, no red.

Must-haves baked into every option:
  • Two logos — Peoples Company lockup + “Proud Member of the International Ag Alliance” (recreated SVG placeholder; swap real vector at build).
  • Phone number, Land Investment Expo link, and FarmWorth — these can sit with the nav or apart; FarmWorth shows as a plain link in some options and a pill button in others so you can compare both treatments.
  • Social icons — Facebook · X · Instagram · LinkedIn · YouTube (as on the current site’s top bar).
  • Four mega-menus: Real Estate · The Company · What We Do · Insights & News (each has a sub-nav pop-down, not shown in this layout pass).
