Header — design styles

Four style directions for the header we picked. Layout is now held mostly constant so you're judging the look — color, type, buttons, search. Options A–C sit on the Option 07 (Inline Links) layout; Option D sits on the Option 09 (Right-Aligned Compact) layout so the alternate gets styled up too.

Static only — no hover pop-downs or working search; the ▾ just marks a sub-nav exists, and the search control is a visual mock (mechanic noted per option). Colorblind-safe, no red.

Baked into every option (from this round's decisions):
  • Utility bar mostly gone — social icons and the “Proud Member / International Ag Alliance” badge have moved to the footer. Options A–D are one clean row; Option E keeps a slim utility strip (phone top-right) on request, so we can compare with vs. without.
  • “Login” is the account entry (reverted from “FarmWorth”).
  • Expo + Login render as buttons, not plain nav links — separating “navigation” from “actions.”
  • Search added — magnifying-glass control (mechanic varies below).
  • Four mega-menus: Real Estate · The Company · What We Do · Insights & News.
